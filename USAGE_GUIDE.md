# SeeYouThere 사용 가이드

## ✨ 장소 검색 및 선택 방법

### 1. 장소 검색
1. 검색창에 장소 이름 입력 (예: "신림역", "스타벅스 강남점")
2. **엔터키** 또는 **검색 버튼** 클릭
3. 검색 결과 드롭다운 표시

### 2. 장소 선택
- 드롭다운에서 원하는 장소 **클릭**
- 선택한 장소의 **이름**이 검색창에 표시됨
  - ✅ 예: "신림역" 클릭 → "신림역" 표시
  - ❌ 아님: "Seoul, South Korea" 같은 주소 표시

### 3. 검색 결과 형식

```
📍 신림역                    ← 장소 이름 (굵은 글씨)
   서울 관악구 신림동 ...     ← 상세 주소 (작은 글씨)
```

## 🎯 검색 팁

### 정확한 검색을 위한 키워드
- ✅ **역/정거장**: "신림역", "강남역", "오송역"
- ✅ **상호명**: "스타벅스 강남점", "맥도날드 신촌점"
- ✅ **건물명**: "서울시청", "63빌딩", "코엑스"
- ✅ **관광지**: "남산타워", "경복궁", "롯데월드"

### 피해야 할 검색어
- ❌ 너무 일반적: "카페", "음식점" (결과가 너무 많음)
- ❌ 주소만: "서울시 강남구" (장소가 아닌 지역)
- ❌ 좌표: "37.123, 127.456"

## 🚀 사용 흐름

### 참여자 등록
1. **병사 이름**: 참여자 이름 입력
2. **출발 거점**: 
   - 장소 검색 (예: "신림역")
   - 검색 결과에서 선택
   - ✅ 확인: "신림역"이 표시됨
3. **이동 수단**: 대중교통 또는 자동차 선택
4. **병력 추가** 버튼 클릭

### 후보지 등록
1. **목표 지점**:
   - 장소 검색 (예: "강남역")
   - 검색 결과에서 선택
   - ✅ 확인: "강남역"이 표시됨
2. **후보지 추가** 버튼 클릭
3. 자동으로 모든 참여자의 소요시간 계산

### 결과 확인
- **가장 공평한 장소**: 시간 편차가 적은 곳
- **가장 빠른 장소**: 총 소요시간이 짧은 곳
- 각 후보지별 참여자별 소요시간 확인

## 🔧 문제 해결

### "검색 결과가 없습니다"
- 검색어를 더 구체적으로 (예: "역" → "신림역")
- 맞춤법 확인
- 영문/한글 확인

### 주소가 표시되는 경우
- **정상**: 검색 결과 드롭다운에서 주소가 아래에 작게 표시
- **비정상**: 선택 후 검색창에 주소 표시
  - → 장소를 다시 선택해주세요
  - → 여전히 문제가 있다면 페이지 새로고침

### API 오류
- 브라우저 콘솔 확인 (F12)
- Google Maps API 키 확인
- Places API (New) 활성화 확인

## 📱 모바일 사용

- 검색창 탭 → 키보드 입력
- 검색 버튼 탭
- 결과 목록에서 원하는 장소 탭
- 선택된 장소 이름 확인

## 💡 Pro Tips

1. **정확한 장소명 사용**: "역", "점", "관" 등 접미사 포함
2. **중복 검색 방지**: 로딩 중에는 버튼이 비활성화됨
3. **후보지 비교**: 여러 장소 추가 후 통계 비교
4. **공유 기능**: 결과 확인 후 링크 공유로 팀원과 공유

- 개발자 도구 콘솔 확인
